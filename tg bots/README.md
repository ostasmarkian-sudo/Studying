# Python practice projects

This workspace contains small Python exercises and two Telegram bots.

## Layout

- `bots/` — Telegram bot projects
- `exercises/` — standalone Python exercises
- `data/` — CSV, database, and Minecraft manifest data
- `drafts/` — unfinished experiments

## Setup

1. Activate `.venv`.
2. Install dependencies: `python -m pip install -r requirements.txt`.
3. Copy `.env.example` to `.env` and add newly generated Telegram bot tokens.
4. Load `.env` in your terminal or VS Code launch configuration before running a bot.

Never commit `.env` or Telegram bot tokens.
